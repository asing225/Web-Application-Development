Instructions to run this project:

1. Open file mysql_phonebook.sql and run the sql queries to setup the database for this app.

2. Open the properties/rdbm.properties file and change the jdbc username and password properties.

3. Create a lib folder in project root folder and add the mysql, jersey, javax-json libraries in this folder.
NOTE: Using lib folder from BookTownREST example will work for this api.

4. Open build.properties file and change the tomcat_webapps property to your local tomcat webapps.

5. Run build.xml using terminal to deploy the project properly.
NOTE: Build.xml might not work using Eclipse, please use command line to deploy build.xml

6. In the project directory go to apidoc folder and open index.html to read the api documentation.

7. Access the api localhost using proper url given in documentation.
